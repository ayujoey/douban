介绍：本项目旨在模仿豆瓣官网

主页实现了搜索功能并可以直接跳转到搜索的电影页面，可翻页更换的随机推荐功能，以及用轮播图播放的热门推荐功能

实现可登录注册并传入后端

实现从后端读取电影id并装配不同的电影详情页

实现个人空间并可以在本地修改自我介绍
